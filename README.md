# mineralmatrix

A vertically integrated mineral-targeting data platform for geospatial data engineering, QA, feature engineering, ML-ready outputs, and drilling feedback simulation.

## Initial stack
- Python
- GeoPandas
- GDAL
- Rasterio
- PostgreSQL + PostGIS
- FastAPI
- QGIS
- GitHub Pages
- Render
- Object storage

## Planned modules
- ingestion
- schema_registry
- validation
- standardization
- spatial_processing
- warehouse
- feature_engineering
- modeling
- drill_feedback
- api
- frontend
- docs

## Quick start

```bash
python -m pip install -U pip
pip install -e .[dev]
pytest
```
